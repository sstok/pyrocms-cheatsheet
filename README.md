## Install

Clone this repo and install all Node.js packages, and configure Semantic-ui for the project.

``` bash
# install dependencies
./scripts.sh install
```

## Build

The build process will populate the `public` folder with javascript and css files.

``` bash
# compile resources
gulp
```
